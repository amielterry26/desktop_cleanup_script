I created a little script for both Mac and Windows users that automatically organizes your desktop — moving files into a tidy folder for you to sort through later (or not… no judgment 😄).

Below you’ll find quick, simple instructions along with a demo to show exactly how it works. The scripts are attached — feel free to run it and enjoy the calm.

🛠 For WINDOWS USERS:

  1. Download the .ps1 files:
    • desktop_cleanup.ps1
    • desktop_restore.ps1
  2. Open PowerShell and navigate to your Desktop. Example:
cd "C:\Users\YOURNAME\OneDrive - Seitel Systems, LLC\Desktop"

🔁 Replace YOURNAME with your own Windows username

  1. Run:
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
./desktop_cleanup.ps1

Your desktop is now clean! Use desktop_restore.ps1 to undo if needed.

🍏 Mac Users (You studs — proud of you 😎)

  1. Download the .sh files:
    • desktop_cleanup.sh
    • desktop_restore.sh
  2. Open Terminal and run:
chmod +x ~/Desktop/desktop_cleanup.sh
./desktop_cleanup.sh

💬 Any questions, feel free to reach out. (Also if you don't like it and want to restore to your messy ways, you can run the desktop_restore.ps1 or desktop_restore.sh to reset.)
